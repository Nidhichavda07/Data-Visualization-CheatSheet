#  Data Visualization Cheat Sheet Project

A complete data visualization project built using Python and Google Colab.

This project demonstrates 12 different types of charts using a real-world Air Quality dataset (Pakistan - Faisalabad).

The goal of this project is to understand how different visualizations represent data insights and how to use them properly in real-world analytics.

---

---

##  Dataset Information

The dataset contains hourly air quality measurements with the following features:

- timestamp  
- city  
- latitude  
- longitude  
- pm10  
- pm2_5  
- carbon_monoxide  
- nitrogen_dioxide  
- sulphur_dioxide  
- ozone  
- dust  
- temperature  
- humidity  
- precipitation  
- wind_speed  
- wind_direction  
- pressure  
- date  
- hour  
- day_of_week  
- month  
- month_name  
- year  
- is_weekend  
- season  
- aqi_category  

The data represents pollution levels recorded hourly in Faisalabad, Pakistan.
